ColorBuf Class

Extends Component
Defined in: src/core/colorBuf.js:6
Module: XEO

A ColorBuf configures the WebGL color buffer for attached GameObjects.

  • A ColorBuf configures the way that pixels are written to the WebGL color buffer, and is not to be confused with ColorTarget, which holds the final pixel colors so that they may be fed into Textures.

Example

In this example we're configuring the WebGL color buffer for a GameObject.

The scene contains:

  • a ColorBuf that enables blending and sets the color mask,
  • a Geometry that is the default box shape and
  • a GameObject attached to all of the above.
var scene = new XEO.Scene();

var colorBuf = new XEO.ColorBuf(scene, {
   blendEnabled: true,
   colorMask: [true, true, true, true]
});

var geometry = new XEO.Geometry(scene); // Defaults to a 2x2x2 box

var gameObject = new XEO.GameObject(scene, {
   colorBuf: colorBuf,
   geometry: geometry
});
