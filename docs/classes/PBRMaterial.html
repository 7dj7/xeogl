PBRMaterial Class

Extends Material
Defined in: src/core/pbrMaterial.js:1
Module: XEO

A PBRMaterial is a Material subclass which defines the appearance of attached GameObjects using a physically-based rendering model.

Overview

Physically Based Rendering (PBR) is a method of shading and rendering that provides a more accurate representation of how light interacts with surfaces. It can be referred to as Physically Based Rendering (PBR) or Physically Based Shading (PBS). Depending on what aspect of the pipeline is being discussed, PBS is usually specific to shading concepts and PBR specific to rendering and lighting. However, both terms describe on a whole, the process of representing assets from a physically accurate standpoint. - Wes McDermott, Allegorithmic PBR Guide, Vol. 2

See Shader Inputs for the variables that PBRMaterials create within xeoEngine's shaders.

Example 1: Metallic material with color map

In this example we have

  • a Texture,
  • a metallic PBRMaterial which applies the Texture as a color map,
  • a Geometry that is the default box shape, and
  • a GameObject attached to all of the above.

Note that the value for the PBRMaterial's color property is ignored and redundant, since we assign a Texture to the PBRMaterial's colorMap property. The Texture's pixel colors then directly provide the metallic base color of each fragment across the Geometry surface.

var scene = new XEO.Scene();

var colorMap = new XEO.Texture(scene, {
   src: "colorMap.jpg"
});

// A metallic material
var metalMaterial = new XEO.PBRMaterial(scene, {

   // Metallic material (else specular)
   metallic: true,

   // Base color in Metallic mode (else albedo in Specular mode)
   color: [1.0, 0.8, 0.0],
   colorMap: colorMap,

   // Black (0.0) for non-metal and white (1.0) for raw metal
   metalness: 0.8,

   // Fresnel values
   specularF0: 0.0,
});

var geometry = new XEO.Geometry(scene); // Geometry without parameters will default to a 2x2x2 box.

var object = new XEO.GameObject(scene, {
   material: metalMaterial,
   geometry: geometry
});

    Parent Scene, creates this PBRMaterial within the default Scene when omitted

  • [cfg] optional

    The PBRMaterial configuration

    • [id] String optional

      Optional ID, unique among all components in the parent Scene, generated automatically when omitted.

    • [meta=null] String:Object optional

      Metadata to attach to this PBRMaterial.

    • [metallic=false] Boolean optional

      Selects the material mode: Metallic when true, Specular otherwise (default).

    • [color=[ 1.0, 1.0, 1.0 ]] Array of Number optional

      Base color in Metallic mode, albedo in Specular mode.

    • [colorMap=null] Texture optional

      A color map Texture, which will override the effect of the color property. Must be within the same Scene as this PBRMaterial.

    • [emissive=[ 1.0, 1.0, 1.0 ]] Array of Number optional

      Emissive color.

    • [emissiveMap=null] Texture optional

      An emissive map Texture, which will override the effect of the emissive property. Must be within the same Scene as this PBRMaterial.

    • [opacity=1] Number optional

      Scalar in range 0-1 that controls opacity, where 0 is completely transparent and 1 is completely opaque. Only applies while Modes transparent equals true.

    • [opacityMap=null] Texture optional

      An opacity map Texture, which will override the effect of the opacity property. Must be within the same Scene as this PBRMaterial.

    • [roughness=0] Number optional

      Scalar in range 0-1 that controls roughness, where 0 is 100% glossiness and 1 is 100% roughness.

    • [roughnessMap=null] Texture optional

      A roughness map Texture, which will override the effect of the roughness property. Must be within the same Scene as this PBRMaterial.

    • [bumpMap=null] Texture optional

      A bump map Texture. Must be within the same Scene as this PBRMaterial.

    • [metalness=1] Number optional

      Scalar in range 0-1 that controls how much metalness is applied for non-metalic PBRMaterials; black (0.0) for non-metal and white (1.0) for raw metal.

    • [specularF0=1] Number optional

      Scalar in range 0-1 that controls how much Fresnel is applied for non-metalic PBRMaterials.

    • [specular=[ 1.0, 1.0, 1.0 ]] Array of Number optional

      Specular color.

    • [specularMap=null] Texture optional

      A specular map Texture, which will override the effect of the specular property. Must be within the same Scene as this PBRMaterial.
