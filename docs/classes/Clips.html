Clips Class

Extends Component
Defined in: src/core/clips.js:6
Module: XEO

A Clips is a group of arbitrarily-aligned Clip planes that clip fragments of attached GameObjects.

  • Each Clip is specified in World-space, as being perpendicular to a vector dir that emanates from the origin, offset at a distance dist along that vector.
  • You can move each Clip back and forth along its vector by varying its dist.
  • Likewise, you can rotate each Clip about the origin by rotating its dir.
  • Each Clip is has a mode, which indicates whether it is disabled ("disabled"), discarding fragments that fall on the origin-side of the plane ("inside"), or clipping fragments that fall on the other side of the plane from the origin ("outside").
  • You can update each Clip's mode to activate or deactivate it, or to switch which side it discards fragments from.

Example

  • In this example we have a GameObject that's clipped by a Clips that contains two Clip planes.
  • The first Clip plane is on the positive diagonal, while the second is on the negative diagonal.
  • The GameObject's Geometry is the default 2x2x2 box, and the planes will clip off two of the box's corners.
var scene = new XEO.Scene();

// Clip plane on negative diagonal
var clip1 = new XEO.Clip(scene, {
   dir: [-1.0, -1.0, -1.0], // Direction of Clip from World space origin
   dist: 2.0,               // Distance along direction vector
   mode: "outside"          // Clip fragments that fall beyond the plane
});

// Clip plane on positive diagonal
var clip2 = new XEO.Clip(scene, {
   dir: [1.0, 1.0, 1.0],
   dist: 2.0,
   mode: "outside"
});

// Group the planes in a Clips
var clips = new XEO.Clip(scene, {
   clips: [
       clip1,
       clip2
   ]
});

// Geometry defaults to a 2x2x2 box
var geometry = new XEO.Geometry(scene);

// Create an GameObject, which is a box sliced by our clip planes
var object = new XEO.GameObject(scene, {
   clips: clips,
   geometry: geometry
});
