xeoEngine

A sleek and powerful 3D engine for WebGL

Download this project as a .zip file Download this project as a tar.gz file

About

xeoEngine is a 3D engine for WebGL from @xeolabs.

  • Flexible object-component API
  • Defaults everywhere, providing training wheels for learning the API
  • Fully runtime-editable scene
  • Optimised - draw list caching, state sorting etc.
  • Save and load scenes as JSON

Usage

Simply include xeoengine.js in your project to begin creating 3D worlds!

Concept

Build a 3D scene from components:

// xeoEngine provides implicit defaults for everything, which provides
// training wheels for learning the api. This example is using all sorts
// of defaults, such as cameras, light sources etc.

var scene = new XEO.Scene();

var material = new XEO.Material(scene, {
    diffuse: [ 0.6, 0.6, 0.7 ]
});

// A 2x2x2 box by default
var geometry = new XEO.Geometry(scene);

var object = new XEO.GameObject(scene, {
    material: myMaterial,
    geometry: myGeometry
});

Update a scene by setting component properties:

// Everything in xeoEngine fires change events
 material.on("diffuse",
     function(value) {
            console.log("Value of material diffuse is now: " + value);
     });

 // Read and write properties directly
 // This is going to fire our change listener above
 material.diffuse = [0.9, 0.9, 0.6];

Dynamically edit a scene at runtime:

  // Everything in your scene is editable
  var texture = new XEO.Texture(scene, {
      src: "myTexture.jpg"
  });

  material.diffuseMap = texture;

Save and load the scene as JSON:

 var json = scene.json;

 var scene2 = new XEO.Scene({ json: json });
