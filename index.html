xeoEngine

A sleek 3D rendering API for WebGL

Download this project as a .zip file Download this project as a tar.gz file

About

xeoEngine is a WebGL-based 3D engine from @xeoLabs

Features

  • Define a scene as a simple component-object graph
  • Use sensible defaults for everything, ie. convention over configuration
  • Dynamically redefine anything in a scene at runtime
  • Render large scenes efficiently
  • Create a wide range of 3D effects
  • Export and import scenes as JSON
  • Complete API docs

Usage

Simply include xeoengine.js in your project to begin creating 3D worlds!

Quick Start

Define a 3D scene as a simple component-object graph, as shown in this example. Since xeoEngine provides defaults for everything, only specify what you need to override. Our example uses all sorts of defaults, such as the camera, lights, even a default canvas.

var scene = new XEO.Scene();

var material = new XEO.Material(scene, {
    diffuse: [ 0.6, 0.6, 0.7 ]
});

// A 2x2x2 box by default
var geometry = new XEO.Geometry(scene);

var object = new XEO.GameObject(scene, {
    material: myMaterial,
    geometry: myGeometry
});

Animate your scene by setting component properties. All properties fire change events which you can subscribe to.

material.on("diffuse",
     function(value) {
            console.log("Value of material diffuse is now: " + value);
     });

 // This is going to fire our change listener above
 material.diffuse = [0.9, 0.9, 0.6];

Redefine anything in your scene dynamically, at runtime. Create, destroy, link and unlink components, or set any component property.

  var texture = new XEO.Texture(scene, {
      src: "myTexture.jpg"
  });

  material.diffuseMap = texture;

Save and load your scene as JSON.

 var json = scene.json;

 var scene2 = new XEO.Scene({ json: json });
